# BOM
* 75x Gateron Milky Yellow Switches
* 75x White Keycaps (all normal sized)
* 1x PCB w/ bottom-side PCBA due to bare RP2040, tiny capacitors, resistors, and USB-C port
* 1x White 3D Printed Case
* 75x Diodes
